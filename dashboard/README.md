# Gemini Flash TTS Sample Dashboard

Static HTML page with all the audio samples generated by `scripts/demo_audio_tags.py`, `scripts/demo_voices.py`, and `scripts/demo_multi_speaker.py`.

## Preview locally

```bash
open dashboard/index.html
```

## Deploy to Vercel

```bash
cd dashboard
vercel --prod
```

Or, from the repo root, point Vercel at the `dashboard/` directory as the root output.

## Updating samples

When new samples are generated by the demo scripts (they write to `data/samples/`), sync them in:

```bash
cp data/samples/*.wav dashboard/samples/
```

Then redeploy.
